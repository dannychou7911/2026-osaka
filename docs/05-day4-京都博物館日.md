## 景點詳細

### 京都鐵道博物館
- 日本最大鐵道博物館
- 必看：
  - 蒸汽火車 SL 體驗（額外付費，可實際搭乘）
  - 駕駛模擬器（小孩超愛）
  - 扇形車庫（20 輛蒸汽火車展示）
  - 3 樓展望台（看新幹線經過）
- 票價：成人 ¥1,500 / 中高生 ¥1,300 / 小學生 ¥500 / 3歲以上 ¥200
- 嬰兒車可推入

### 京都水族館
- 距離鐵道博物館步行 7 分鐘
- 室內冷氣涼爽
- 必看：
  - 海豚秀（每日 3-4 場）
  - 山椒魚（特別展示）
  - 企鵝區
- 票價：成人 ¥2,400 / 高中生 ¥1,800 / 中小生 ¥1,200 / 3歲以上 ¥800

### 京都車站
- **拉麵小路**：日本各地名拉麵聚集（10 家店家）
- **伊勢丹百貨**：地下美食街、伴手禮齊全
- **空中花園**：免費景點，可俯瞰京都市區
