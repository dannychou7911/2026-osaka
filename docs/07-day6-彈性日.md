# Day 6｜7/29 (三) 彈性日 ⏸️

---

## 🅰️ 方案 A：京都嵐山一日遊 🎋（推薦）

### 行程概要

| 時段 | 行程 | 備註 |
|------|------|------|
| 08:30 | 大阪 → 京都嵐山（約 1 小時） | |
| 10:00-11:30 | **嵯峨野觀光小火車** | 須提前預訂 |
| 11:30-13:00 | **竹林小徑、渡月橋** | 拍照打卡 |
| 13:00-14:30 | 嵐山午餐 | 湯豆腐、抹茶 |
| 14:30-16:00 | **天龍寺** | 世界文化遺產 |
| 17:00 | 返回大阪 | |
| 18:00-21:00 | **黑門市場** | 海鮮燒烤晚餐 |

### 景點詳細

#### 嵯峨野觀光小火車 Sagano Romantic Train
- 沿保津川峽谷行駛，25 分鐘
- 票價：成人 ¥880 / 6歲以下免費
- **超熱門，務必提前 1 個月預訂**
- 推薦 5 號車廂（無玻璃窗，視野最好但風大）

#### 竹林小徑 + 渡月橋
- 京都最具代表性風景
- **早上光線最美**（建議避開人潮）
- 嬰兒車可推

#### 天龍寺
- 世界文化遺產
- 庭園「曹源池」絕美
- 票價：成人 ¥500 / 中小生 ¥300

#### 黑門市場
- 「大阪的廚房」
- 必吃：和牛串燒、海鮮丼、神戶牛壽司
- 8 人團體建議**站著吃**或**找預約制餐廳**

### 待辦事項
- [ ] 預訂嵯峨野小火車（行前 1 個月）
- [ ] 嵐山午餐餐廳預約
- [ ] 黑門市場用餐策略

| 時間 | 行程 |
|------|------|
| 08:00 | 抵達 USJ |
| 全天 | 補玩沒玩到的設施、表演、購物 |

- 集中火力玩 Day 3 沒玩到的設施
