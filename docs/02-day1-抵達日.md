# Day 1｜7/24 (五) 抵達日 🛬

## 行程概要

| 時間 | 行程 |
|------|------|
| 18:05 | 關西機場 KIX 抵達 |
| 18:05-19:30 | 入境、領行李、取預訂票券、買 ICOCA |
| 19:30 | 搭乘 HARUKA / Rapi:t 進市區（依住宿區） |

## 機場交通選擇

### 🟦 HARUKA（JR 特急）
- 直達車站：天王寺 / 新大阪 / 京都
- 票價約 ¥2,400-2,900
- 推薦套票：**ICOCA & HARUKA 套票**（單程 ¥3,200 含 ICOCA）

### 🟥 Rapi:t（南海ラピート）
- 直達車站：**南海難波**
- 票價約 ¥1,490
- KKday/Klook 預訂約 ¥1,300
