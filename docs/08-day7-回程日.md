# Day 7｜7/30 (四) 回程日 ✈️

## 行程概要

| 時間 | 行程 |
|------|------|
| 08:00 | 飯店早餐 |
| 08:30-09:30 | 整理行李 |
| 09:30-10:30 | 飯店附近最後採買（藥妝、伴手禮） |
| 10:30 | Check-out |
| 10:45 | 搭乘 HARUKA / Rapi:t 前往機場 |
| 11:30-12:00 | 抵達關西機場 |
| 12:00-13:30 | Check-in、退稅、安檢 |
| 14:00 | 班機起飛 ✈️ |

## 重點時間計算

### 國際線回程建議
- 起飛前 **2.5 小時抵達機場**（國際線標準）
- 14:00 起飛 → 11:30 前必須抵達 KIX
- 大阪市區 → 關西機場 約 50 分鐘
- → **10:30 前必須出發**

### 退稅注意
- 退稅櫃台在出境後管制區外
- 排隊可能需要 30-60 分鐘
- **務必預留時間**

## 行李處理方案

### 方案 1：當日帶行李去機場（傳統）
- 直接帶到飯店 → 機場
- 拖行李移動較辛苦（特別是帶小孩）

### 方案 2：黑貓宅急便寄機場（推薦）⭐
- 前一天傍晚從飯店寄出
- 隔天直接到機場領取
- 費用：約 ¥2,000-3,000/件
- 適合採購較多伴手禮的家庭

### 方案 3：機場提前 Check-in
- 部分航空公司支援關西機場前一日 Check-in
- 較少航空公司有此服務，需查詢

## 待辦事項

- [ ] 確認回程航空公司行李規定
- [ ] 預訂機場交通票券（HARUKA / Rapi:t 回程票）
- [ ] 整理需要退稅的收據
- [ ] 確認伴手禮採購清單
- [ ] 黑貓宅急便寄送（如需要）

## 最後採買清單建議

### 藥妝
- 鼻通、撒隆巴斯、合利他命、休足時間、ROIHI 貼布
- 推薦藥妝店：松本清、大國藥妝、Don Quijote

### 食品伴手禮
- 白色戀人、ROYCE 巧克力、東京香蕉
- LeTAO 起司蛋糕（KIX 機場有店面）
- 北海道牛奶餅乾

### 小孩用品
- 卡通造型零食（USJ、迪士尼、寶可夢周邊）
- 文具（無印良品、伊東屋）

## 機場必去

- **大阪/關西機場 LeTAO 起司蛋糕專賣店**
- **PABLO 半熟起司塔**
- **551 蓬萊豬肉包**（限時販售）

## 注意事項

- **不要排任何景點**，純休息日
- 確認所有護照、登機證、貴重物品
- 確認小孩的隨身飲水與零食
- 早一點到機場，避免排隊壓力
- 退稅商品不可拆封（海關抽查）
